# Product image slots

Replace the preview visuals by adding product images in this folder and updating
the `media.mainImage`, `media.gallery`, and `media.colorImages` fields in
`data/products.json`.

Recommended filenames:

- `fold-main.webp`
- `fold-01.webp`
- `fold-02.webp`
- `s1-main.webp`
- `s1-01.webp`
- `s1-02.webp`
- `noma-main.webp`
- `noma-01.webp`
- `noma-02.webp`
- `mini-main.webp`
- `mini-01.webp`
- `mini-02.webp`

Recommended main image size: 1600 x 1200 pixels or larger.
